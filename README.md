# Flask-notes-mini-project
A simple and secure web-based Notes application that allows users to sign up,
log in, add notes, view notes, and delete notes while storing all information securely
in a database. Demonstrates authentication, CRUD operations, and database handling.

Features:
- User registration
- Secure login
- Add, delete, and view notes
- Protected pages
- Database integration
- AJAX delete functionality

- What You Learn:
Flask routing, templating, Bootstrap UI, ORM, authentication, CRUD operations,
AJAX handling, and overall full-stack web development basics.
